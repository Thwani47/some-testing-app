Python Flask application deployed via Kratix

- Flask REST API
- Health check endpoint
- Containerized with Docker
- CI/CD with GitHub Actions

## Redis Integration

This app is connected to Redis for:
- Visit counting
- Session storage
- Caching

### Endpoints
- `GET /count` - Increment and return visit count
- `GET /health/redis` - Check Redis connection health


## Local Development

```bash
# Install dependencies

# Run the app
python app.py
```

The app will be available at `http://localhost:5000`

- `GET /health` - Health check
